# Sentiment Analysis using BERT

## Overview

This project classifies IMDB movie reviews into **positive** or **negative** sentiments using a **fine-tuned BERT model**. It demonstrates advanced NLP techniques using transformer architectures.

---

## Features

* Fine-tuned **BERT (transformer model)**
* Handles text preprocessing and tokenization
* Uses Hugging Face Trainer API
* Evaluated using multiple performance metrics

---

## Tech Stack

* Python
* PyTorch
* Hugging Face Transformers
* Scikit-learn
* Pandas & NumPy

---

## Workflow

1. Load IMDB dataset
2. Preprocess text data
3. Tokenize using BERT tokenizer
4. Fine-tune BERT model
5. Evaluate performance

---

## Model Performance

* **Accuracy:** 93%
* **Precision:** 93%
* **Recall:** 93%
* **F1 Score:** 93%
